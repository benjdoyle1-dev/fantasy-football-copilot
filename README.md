> # 🏈 Fantasy Football Copilot

An AI-powered fantasy football assistant that helps users make smarter lineup decisions using player projections, matchup analysis, weather, injuries, and natural language explanations.

## Vision

Fantasy Football Copilot is designed to act like a knowledgeable fantasy football advisor.

Instead of simply showing rankings, it explains *why* a player should start or sit and eventually will sync directly with platforms like Sleeper, ESPN, and Yahoo Fantasy.

The goal is to help users make confident lineup decisions in seconds.
